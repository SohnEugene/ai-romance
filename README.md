## 작업 방식
- git pull
- twine에 "sugar cube test.html"을 import
- twine 내부에서 패시지 추가, 수정, 태그 추가 등의 작업 수행
- "Publish to file"로 다운로드
- 레포지토리 내부 sugar cube test.html 업데이트
- instruction.txt를 따라서 js, css 연결
- live server로 잘 작동하는지 확인
- git add .
- git commit -m "작업 내용"
- git push
