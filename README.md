### 주제: AI
### 작품명: 새내기인 내 연애코치가 이세계에선 공략대상?!

## 플레이 방법
- 코드 파일을 로컬로 다운로드
- ZIP파일 압축 해제
- 압축 해제된 폴더를 열어 'dating sim.html' 파일 실행

## 작업 방식
- git pull
- twine에 "dating sim.html"을 import
- twine 내부에서 패시지 추가, 수정, 태그 추가 등의 작업 수행
- "Publish to file"로 다운로드
- 레포지토리 내부 "dating sim.html" 업데이트
- instruction.txt를 따라서 js, css 연결
- live server로 잘 작동하는지 확인
- git add .
- git commit -m "작업 내용"
- git push
